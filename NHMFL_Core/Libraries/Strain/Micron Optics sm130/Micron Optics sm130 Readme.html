3. Known Issues

To report issues or provide feedback about this instrument driver, please send an email to instrument.drivers@ni.com.

Setup Instructions
  • Open National Instruments Measurement & Automation Software.
  • Expand My System and right click on Devices and Interfaces >> Create New.
  • Select VISA TCP/IP Resources >> Next >> Manual Entry or Raw Socket
  • Enter the IP address of sm130 (default IP address is 10.0.0.126) and Port Number 1852 >> Next
  • *Optional Enter an Alias for the TCP/IP raw socket and click Finish
  • The TCP/IP raw socket alias should show up on VISA Resource Control.
